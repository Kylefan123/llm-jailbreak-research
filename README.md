# LLM Jailbreak Research

Research project investigating adversarial prompting and social-engineering-inspired jailbreak attacks against Large Language Models (LLMs).

## Overview

This research explores how trust-building, psychological framing, and token obfuscation techniques can be used to bypass safety-aligned LLM safeguards.
